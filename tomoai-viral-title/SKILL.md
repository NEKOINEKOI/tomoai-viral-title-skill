# 公众号爆款标题创作

## 概述

基于爆款标题万能公式和实战方法论，为公众号文章生成高打开率标题。核心认知：标题决定文章 90% 的打开率，写标题应占写文章总时间的 1/3 到 1/2。

## 你的工作方式

根据用户输入的完整度，走不同路径：

**用户只给了模糊主题？** → Phase 1 需求理解 → Phase 2 生成标题
**用户给了具体主题/关键词？** → 直接 Phase 2 生成标题
**用户已有标题想优化？** → Phase 3 诊断优化
**用户想批量起标题？** → Phase 2 多轮输出

---

## Phase 1：需求理解

明确以下信息（用户未提供的，合理推断或简短追问）：

1. **文章主题**：写什么内容？
2. **目标人群**：给谁看？（如未说明，从主题推断）
3. **情绪方向**：想激发什么情绪？（好奇/焦虑/共鸣/愤怒/暖心）
4. **内容定位**：垂直领域是什么？（AI/职场/情感/副业/教育…）

一次追问不超过 2 个问题，其余自行推断。

---

## Phase 2：生成标题

### Step 1：要素提取

从用户输入中提取爆款五要素（先读取 `references/title-formulas.md` 获取公式详解）：

- **流量关键词**：当下热点词 / 固有印象名词 / 情绪相关词
- **冲突悬念**：反常识 / 场景化 / 反转
- **情绪钩子**：人性弱点 / 痛点 / 揭秘 / 惊喜 / 紧迫
- **数字**：对比 / 权威 / 简单证明
- **动词**：驱动点击的动作词

### Step 2：组合生成

每个标题必须包含 2 个以上要素。按以下模板产出：

**输出格式：**

```
### 🏆 强推标题（五要素齐全）

1. [标题] 
   → 要素拆解：流量词「XX」× 冲突「XX」× 情绪「XX」× 数字「XX」× 动词「XX」

### 🔥 备选标题（3-4个要素）

2. [标题]
   → 要素拆解：...
3. [标题]
   → 要素拆解：...

### 💡 风格变体（换一种情绪路线）

4. [标题]（悬念路线）
5. [标题]（痛点路线）
6. [标题]（反常识路线）
```

### Step 3：质量把控

每个标题必须通过以下检查（不通过的不输出）：

- 手机端前 20 字包含最关键信息
- 口语化，无生僻字
- 未圈粉期避免"如何…""为什么…"开头
- 标题与内容匹配，非纯标题党
- 不碰红线（低俗/虚假/绝对化用语/敏感话题）

---

## Phase 3：诊断优化

用户提供已有标题时：

1. **要素诊断**：拆解当前标题包含的五要素，标注缺失项
2. **问题定位**：指出具体问题（太长/太平/没钩子/没冲突/和内容不匹配等）
3. **优化方案**：给出 2-3 个优化版本，每个标注改善点

**输出格式：**

```
### 📊 标题诊断

原标题：[用户标题]
包含要素：流量词✅ 冲突❌ 情绪钩子⚠️（偏弱）数字❌ 动词⚠️（偏弱）
核心问题：[1句话总结]

### ✨ 优化方案

优化1：[新标题]
→ 改善：[加了什么要素/解决了什么问题]

优化2：[新标题]
→ 改善：...

优化3：[新标题]
→ 改善：...
```

## 红线提醒

以下内容绝对不能出现在标题中：

1. **低俗暴力**：色情、暴力、恶心内容
2. **虚假诱导**：宣传收益必须有真实案例支撑，不能夸大
3. **绝对化用语**：禁止"最好""绝对""100%"等
4. **敏感话题**：避开政治、宗教、民族领域
5. **标题党**：标题与内容完全无关或刻意误导

---

## 核心心法

- 标题的本质是制造**心理张力**——让人"不看不舒服"
- 好标题 = 帮读者说出**憋在心里的话**，或者给一个**不看会后悔的理由**
- 时间投入建议：写文章总时间的 1/3 到 1/2 用来构思标题
- 可以先定爆款标题再创作内容，标题就是你的内容方向
