# 爆款标题公式与模板库

## 一、爆款标题万能公式

**爆款标题 = 流量关键词 × 冲突悬念 × 情绪钩子 × 数字 × 动词**

- 理想状态：同时包含 5 个要素 → 超级爆款
- 实际标准：包含 2-3 个要素 → 好标题
- 长度规则：标题越长需包含的要素越多；手机端显示建议 20 字以内，关键信息放前面

### 五要素详解

| 要素 | 定义 | 技巧 | 示例 |
|------|------|------|------|
| **流量关键词** | 用户天生关心或近期热议的词语 | 三类：①当下热点（微博热搜/知乎热榜/微信指数）②固有印象名词（清华北大/大厂/学区房）③情绪相关词（涨薪/被裁/离婚/彩礼） | "DeepSeek爆火后，我卸载了所有国产AI"中的"DeepSeek" |
| **冲突悬念** | 制造反差感、不可思议感，激发好奇 | 三种：①反常识（越省钱越穷）②场景化（每天深夜挤地铁）③反转（被开除后赚到第一桶金） | "奋斗三年，面试三小时，一秒钟挂了" |
| **情绪钩子** | 激发强烈情绪反应，引发共鸣 | 五类：①人性弱点（贪婪/恐惧/懒惰）②痛点挖掘（社会压力/亲情遗憾）③行业揭秘（健身教练不敢说的减肥法）④惊喜体验（极致赞美/感官冲击）⑤紧迫行动（限时/限量/错过不再） | "为什么累死的总是普通人？" |
| **数字** | 增强内容可信度和具体感 | 三种：①数字对比（月薪五千到三万）②增强权威（从业十年编辑的过稿心得）③证明简单（三分钟学会AI写PPT） | "连续七天做这件事，睡眠质量提升67%" |
| **动词** | 让用户感觉有事发生，驱动点击 | 常见动词：倒闭、被裁、离婚、涨薪、启动、卸载 | "八千万美金的语音输入法被我卸载了"中的"卸载" |

### 平台流量关键词差异

- **公众号/视频号**：职场（离职/裁员）、教育（学区房/鸡娃）、情感（彩礼/婆媳）
- **知乎**：认知偏差、信息茧房、算法伦理、东西方文化差异
- **小红书**：微胖穿搭、早八通勤、极简家居、女性成长

## 二、标题钩子四型

| 钩子类型 | 原理 | 示例 |
|----------|------|------|
| **反常识钩子** | 颠覆常识判断，用户必须点进去看怎么圆 | "有钱男人喜欢丑女人""富二代比你还努力""以结婚为目的的谈恋爱是耍流氓" |
| **痛点直击钩子** | 标题本身就是一刀，插进读者最在意的地方 | "连陪孩子时间都没有，你成功个屁""25岁，名校毕业，年薪30万，猝死" |
| **悬念钩子** | 场景感极强，结果高度意外 | "那个职场霸凌我的主管，被我炒了""半夜点外卖，我收到了一个男朋友" |
| **数字具体化钩子** | 具体数字制造真实感 | "我暴瘦30斤的秘籍""月薪5万的女人私生活是什么样的" |

**共同本质**：制造心理张力——一种不点进去会难受的感觉。

## 三、垂直领域标题模板

| 模板类型 | 公式 | 示例 |
|----------|------|------|
| **人群痛点模板** | 人群 + 痛点场景 + 工具数量 + 效果承诺 | "跨境电商卖家必看：3个AI选品工具，爆款率提升200%""设计师杜绝加班：5个免费AI工具，甲方一次通过" |
| **技术领域模板** | 反常识结论 + 技术名词 + 生活化类比 | "大模型根本不懂语言？用煮面条的原理解释LLM""Stable Diffusion揭秘：像搭乐高一样生成图片" |
| **副业赚钱模板** | 时间紧迫性 + 收益证明 + 操作简易度 | "抓住视频号AI带货红利：单日赚500的保姆教程""小白可上手：用ChatGPT批量生成文案，7天起号实测" |
| **行业预测模板** | 时间节点 + 颠覆性预测 + 行动指令 | "2025年AI编程将取代写手，现在布局这3个技能还来得及""OpenAI下一代模型曝光，抓住机会的4个切入点" |
| **数字+情感+反转** | 数字 + 情感词 + 反转结果 | "结婚3年，我才发现婆婆偷偷攒了10万给小叔子" |
| **热点+观点+争议** | 热点事件 + 鲜明立场 + 争议点 | "'职场PUA'刷屏：真正厉害的人，从不忍气吞声" |
| **故事+情感+疑问** | 故事场景 + 情感冲击 + 疑问收尾 | "凌晨3点被老公赶出家门：婚姻里最可怕的是什么？" |

## 四、通用爆款标题句式库

### 高打开率句式

1. **一个……结束了/彻底结束了**："一个时代结束了""一个赚钱时代彻底结束了"
2. **……必看/必知**："AI从业者必看：这个工具正在取代你的工作"
3. **原来……一直/都**："原来我们一直用错了XX""原来这些人一直在偷偷XX"
4. **千万别……**："千万别在XX上花钱了，这个免费工具更好"
5. **为什么……总是**："为什么累死的总是普通人？"
6. **……的人，都**："年入百万的人，都做对了这3件事"
7. **从X到X，只用了**："从月薪五千到三万，只用了三个月"
8. **X年/X天/X次**："创业5年，我终于想明白了这3件事"
9. **我卸载了/我删除了/我退出了**："我卸载了所有国产AI，替代它的还免费"
10. **没人告诉你/没人敢说**："没人告诉你，XX行业的真相有多残酷"

### 情绪路线句式

- **愤怒/不满**："连XX的时间都没有，你XX个屁"
- **焦虑/紧迫**："再不XX就来不及了"
- **委屈/共鸣**："为什么XX的总是我/普通人"
- **好奇/惊讶**："XX的背后，藏着XX"
- **暖心/治愈**："最高级的XX是XX"

## 五、热点切入公式

**热点操作公式**：热点事件 + 读者强烈情绪 + 明确立场 = 爆款

关键原则：追热点追的是"情绪"而非"事件"本身。

- 不报道事件，而是抓住读者的情绪加以放大
- 不是就事论事，而是帮读者出气/说话
- 不停留在事件本身，直接升华到更深层议题

示例：
- 蒋劲夫家暴 → "你支持的不是蒋劲夫，是家暴"（抓住愤怒情绪）
- 俞敏洪言论 → "直男癌别的不行，甩锅很行"（帮读者出气）
- 刘鑫江歌案 → "法律可以制裁凶手，但谁来制裁人性"（升华议题）

## 六、爆款标题自检清单

每个标题产出后，逐项检查：

- [ ] 是否包含 2 个以上万能公式要素？
- [ ] 手机端前 20 字是否包含最关键信息？
- [ ] 是否口语化，无生僻字？
- [ ] 是否避免了"如何…""为什么…"开头（未圈粉期）？
- [ ] 标题和内容是否匹配，不是纯标题党？
- [ ] 是否避开了红线（低俗/虚假/绝对化/敏感/标题党）？
- [ ] 是否制造了"不看不舒服"的心理张力？
- [ ] 如果去掉热点词，标题本身是否仍有吸引力？
